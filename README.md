AI New Tab - Chrome Extension

AI智能新标签页扩展，集成AI助手、热闻天气、快捷工具。

## Features

- AI Assistant - Ask any question
- Today's News - HackerNews top stories
- Weather - Real-time weather info
- Shortcuts - Quick access to popular sites
- Dark theme design

## Installation

1. Download from GitHub Releases
2. Open chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the extension folder

## Configuration

Edit popup.js to add your AI API key for production use.
